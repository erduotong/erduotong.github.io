## 用法
- 用于 `<script>`标签
```html
<script async></script>
```
## 作用
![资源提示符 async&defer.excalidraw](./附件/资源提示符 async&defer.excalidraw.md)
- 在加载脚本的同时，继续执行DOM (异步执行)
- 但是脚本加载完毕后，就会阻塞DOM，开始执行
