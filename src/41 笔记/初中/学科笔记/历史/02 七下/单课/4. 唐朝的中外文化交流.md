# 日本
## 遣唐使
- `目的`: 学习中国的先进文化
- `方式`: 日本派遣使节来到中国，当时称为“遣唐使”
- `结果`: 把唐朝先进的制度、天文历法、文字、典籍、书法艺术、建筑技术等传回日本 `对日本社会的发展产生了深远的影响`
## 鉴真东渡
- `人物`: *鉴真* `扬州大明寺高僧`
- `时间`: 754年 经过6次尝试，成功抵达日本
- `措施`: 传授佛经、传播中国的医药、文学、书法、建筑、绘画等，主持修建了**唐招提寺**
- `意义`: 为中日文化交流作出了卓越的贡献
# 唐与新罗
- 许多新罗商人来到新罗物产居唐朝进口首位
- 新罗效仿唐朝建立政治制度，采取科举制选拔官吏
- 引入中国的医学、天文、历算等科技成就
- 朝鲜半岛的音乐传入中国
# 玄奘西行
- `时间`:  [贞观初年](./2. 从“贞观之治”到“开元盛世”.md#"贞观之治")
- `人物`: 高僧*玄奘*
- `目的`: 西行前往天竺取经
- `路线`: 长安 -> 天竺 -> 长安
- `行为&意义`:
    - 翻译、传播佛教 `为中国佛教发展做出巨大贡献`
    - 根据他的口述，弟子记录成书《大唐西域记》  `是研究中外交流史的珍贵文献`
# 唐与大食
- 大食([阿拉伯帝国](../../05 九上/单课/12. 阿拉伯帝国.md))派使臣与唐朝通好
- 大食的商人到中国经商
- 不少大食人在中国长期定居，有的还在朝廷中任职
# 繁荣原因
- 国家统一、经济繁荣、文化发达，对周边国家的吸引力大
- 水陆交通发达
- **开放的对外政策**
# 特点
- 对外交往范围广
- 海路、陆路并举
- 对外贸易繁盛
- 双向交流
